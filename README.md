# Auto-Invoice Sorter

**Version:** 1.0.0  
**Status:** Development  
**Target Launch:** Q2 2026  

## Overview

Auto-Invoice Sorter is a Google Workspace add-on that automatically identifies, sorts, and renames invoice files in Google Drive based on vendor, date, and amount. It helps field teams and finance departments maintain organized invoice workflows.

## Key Features

### Invoice Detection

- **Smart Recognition**: Identifies invoice files by name patterns and content
- **Multiple Formats**: Supports PDF, images, and common invoice formats
- **Batch Processing**: Processes entire folders at once

### Auto-Sorting

- **Vendor-Based Folders**: Creates vendor-specific folders automatically
- **Date-Based Organization**: Sorts by invoice date within vendor folders
- **Custom Naming**: Renames files with consistent naming convention

### Validation

- **Duplicate Detection**: Identifies duplicate invoices
- **Missing Information**: Flags incomplete invoice metadata
- **Format Validation**: Ensures files meet naming standards
